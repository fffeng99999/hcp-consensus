# HCP-Consensus 部署指南

## 快速开始 (5分钟)

### 1. 克隆项目
```bash
git clone https://github.com/fffeng99999/hcp-consensus.git
cd hcp-consensus
```

### 2. 构建二进制
```bash
make build
```

### 3. 初始化测试网络
```bash
make init
```

### 4. 启动节点
```bash
make start
```

### 5. 验证部署
```bash
make status
```

## 系统要求

- **Go**: 1.22+
- **Docker**: 20.10+
- **Docker Compose**: 2.0+
- **CPU**: 4核
- **内存**: 8GB
- **磁盘**: 20GB
